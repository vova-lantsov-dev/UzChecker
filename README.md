Простий консольний застосунок, що повідомляє про наявність білетів на обрані потяги Укрзалізниці.

### Локальний запуск

Передумови:
- .NET 8 SDK
- PlayWright (виконати скрипт встановлення з офіційного сайту)

Процес:
- Вказати токен для телеграм боту, через який будуть надодити сповіщення
- Змінити конфігурацію `appsettings.json`, вказавши бажаний день, номери поїздів і назви станцій посадки/прибуття, бажаний клас вагону (купе, люкс тощо)
- Запустити програму через IDE чи `dotnet run`

### Публікація на власний сервер через SSH

Передумови:
- VPS з публічною IPv4 адресою і доступним SSH
- Зробити fork репозиторія і вказати Actions Secrets в його налаштуваннях (SSH_USER, SSH_PORT, SSH_PRIVATE_KEY, SSH_HOST, TELEGRAM_BOT_TOKEN)

Процес:
- Змінити `appsettings.json` / `UzChecker.AppHost.csproj` і закомітити ці зміни в будь-яку гілку
- Дочекатися, коли GitHub Actions пайплайн виконається (артефакт буде завантажено у вказаний Docker registry)
- Перейти у мануальний пайплайн Deploy і запустити його на тій же гілці, куди закомічено зміни

Важливі деталі:
1) `SSH_PRIVATE_KEY` повинен бути вставленим 1 в 1 як він зберігається у файлі на комп'ютері; не можна прибирати пустий рядок у кінці файлу.
2) Інформація щодо репозиторія зберігається в `.csproj` файлі. Її треба змінити (окрім базового імеджу).
